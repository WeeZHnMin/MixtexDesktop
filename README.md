# MixtexDesktop

**MixtexDesktop** is a desktop application that allows users to:
- Convert image-based math text (e.g., handwritten or printed equations) into **LaTeX** format
- Convert **LaTeX** code into a **.docx** file compatible with Microsoft Word or WPS Office

It is especially useful for students, researchers, and educators who frequently work with mathematical documents.

---

## ✨ Features

- 🖼️ Image-to-LaTeX conversion using deep learning  
  基于深度学习的图片识别生成 LaTeX 数学公式  
- 📄 LaTeX-to-DOCX conversion (Word/WPS Office compatible)  
  将 LaTeX 文本转换为 Word (.docx) 文档，兼容 Microsoft Office 和 WPS
