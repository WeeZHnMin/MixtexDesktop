Introduction to Overleaf

A collaborative LaTeX editor for scientific writing

📚 What is Overleaf?

Overleaf 是一个基于云端的 LaTeX 编辑平台，专为撰写学术论文、技术文档、简历、幻灯片等科学内容而设计。它允许用户通过网页浏览器直接编写 LaTeX 文档，无需安装任何软件或配置环境。用户可以在线查看编译结果，实现所见即所得的预览体验。

Overleaf is a cloud-based LaTeX editing platform designed for writing academic papers, technical documents, resumes, slides, and other scientific content. It allows users to write LaTeX documents directly in a web browser without the need to install any software or configure an environment. Users can view compilation results online, achieving a WYSIWYG preview experience.

Overleaf 提供了丰富的模板库，支持多种期刊和大学的论文格式，用户可以轻松选择合适的模板开始撰写。此外，Overleaf 还支持多人协作编辑，实时同步修改，极大地提高了团队合作的效率。

Overleaf offers a rich library of templates supporting various journal and university paper formats, allowing users to easily select the appropriate template to start writing. Additionally, Overleaf supports collaborative editing with real-time synchronization of changes, greatly enhancing team collaboration efficiency.

🚀 Features

  • 自动LaTeX编译和预览
  • 期刊和大学论文的内置模板
  • Latex版本选择和论文历史记录

🎯 Why Use Overleaf?

简化了LaTeX编写过程并消除了安装软件的需要。无论是在写论文、期刊文章还是简历，Overleaf都能让它变得简单高效。

📖 How To Use

Open Overleaf Now

注册一个账号，来到主页，在New Project创建一个空白项目。

点击左上角 Menu(菜单) 中找到 Compiler(编译器)，将编译器切换为 XeLaTeX，然后将下方的模板粘贴到 main.tex 。然后点击Recompile(重新渲染编译) 按钮即可看到效果。

编译完成后可将文档导出

查阅Pdf效果
